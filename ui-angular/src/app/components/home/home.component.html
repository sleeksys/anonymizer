Project Description

The goal of this project is to provide an easy to use open source tool for data anonymizing.

In this case, we start building a server side application based on Java Spring Boot. We did not use any complicated user authentication way, only a simple unique context's id that helps to identify a process for a given session HttpSession.

Anyone is free to download the repository and make any changes for personal or commercial use.

Technologies: Programming languages & Frameworks

Back-End:
Java Spring Boot 1 JPA 2 JDBC 2 H2 2 Lombok 3 Apache POI 4

1*: Why Spring Boot? The main goal is to reduce development, unit test and integration time. And a really important point is the ease of develop production ready web applications.

2*: JPA, JDBC, H2 The keyword is Embedded Databases. Our solution is built using this technology that is easy to store and access information. Quick data processing, self contained and low latencies in terms of data process.

3*: Lombok Lombok helps us build powerful Java code without implementing any getter, setter, equals methods. The library builds these and much more automatically and we reduce the code we need to produce. More about Lombok

4*: Apache POI The Java library provides a lot of features for manipulating file formats created by Microsoft Office. In this case, we need it to read and write Microsoft Excel files.

Front-End: for test cases only:
Angular Bootstrap

Angular For our test cases, we need a powerful JavaScript framework providing a set of HTTP functions for REST. So we choose Angular.

Bootstrap No need to build templates and components from scratch. Bootstrap provides the CSS and JS components we wanted to build this test UI.
